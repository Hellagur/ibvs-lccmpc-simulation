# Laguerre-Parameterized Adaptive Image-Based Visual Predictive Control

This is the source code for the paper:  

**English Title:**  
*Laguerre-Parameterized Adaptive Image-Based Visual Predictive Control for Non-Cooperative Spacecraft Rendezvous*

**中文标题：**  
**非合作航天器交会Laguerre参数化自适应图像视觉预测控制**

The paper has been **accepted for publication** in *Journal of Astronautics* (宇航学报), currently in press.

## Dependencies

-   MATLAB R2023a

-   Toolboxes:

    -   Mosek 11.0

    -   Fuzzy Logic Toolbox
